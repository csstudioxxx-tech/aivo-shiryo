==========================================================
 手順A：ネットに公開する（GitHub Pages）※全員が見れるURLを作る
==========================================================
1. パソコンで github.com を開いてログインする。
2. 右上の「＋」→「New repository（新しいリポジトリ）」を押す。
3. Repository name に  aivo-shiryo  と入力。
   「Public」を選ぶ。→ 緑の「Create repository」を押す。
4. 次の画面で「uploading an existing file」という青い文字を押す。
5. このフォルダの中の【HTMLファイル5つ】を、画面にドラッグ＆ドロップする。
   （index.html と heya-sagashi-… の4つ）
6. 下の緑「Commit changes」を押す。
7. 上のメニュー「Settings（設定）」→ 左の「Pages」を押す。
8. 「Branch」の所を  main  ／  /(root)  にして「Save」。
9. 1〜2分待つと、上の方に
      https://（あなたのID）.github.io/aivo-shiryo/
   というURLが出る。← これが一覧ページ。ブックマークして全員に共有。

   ✅ これで完成。商談中はこのURLを開いて使えます。


==========================================================
 手順B：メモを「全員のパソコンで同じ」にする（任意）
==========================================================
※ これをやらないと、メモは各パソコンごとの保存になります。
※ やると、誰かが書いたメモを全員が見られる＆検索できます。

1. script.google.com を開く →「新しいプロジェクト」。
2. 最初から書いてあるコードを【全部消す】。
3. ai-voyage-library-GAS.gs.js をメモ帳で開いて【中身を全部コピー】し、
   さっきの画面に【貼り付け】る。→ 保存（フロッピーのマーク）。
4. 右上「デプロイ」→「新しいデプロイ」。
5. 「種類の選択（歯車）」→「ウェブアプリ」を選ぶ。
6. 「実行するユーザー：自分」「アクセスできるユーザー：全員」にする。→「デプロイ」。
7. 初回は許可を求められる→自分のGoogleで承認。
   （「このアプリは確認されていません」と出たら：詳細 → 安全でないページに移動、で進める）
8. 出てきた「ウェブアプリのURL」（最後が /exec ）を【コピー】。
9. index.html をメモ帳で開く。
   キーボードの Ctrl+F で   GAS_URL:""   を探す。
   "" の間に、コピーしたURLを貼る。↓こうなればOK
      GAS_URL:"https://script.google.com/macros/s/XXXX/exec"
   → 保存する。
10. その index.html を、手順Aと同じやり方で GitHub に上げ直す（置き換え）。

   ✅ これで全パソコンで同じメモ・タグになります。

【コードを直したとき】
  script.google.com で「デプロイを管理」→鉛筆マーク→
  バージョンを「新しいバージョン」にして「デプロイ」。
  ※こうすればURLは変わりません（新規デプロイはURLが変わるのでダメ）。


==========================================================
 毎日の使い方
==========================================================
・公開URL（または index.html）を開く。
・上の検索窓に言葉を入れると、資料名・タグ・メモから絞り込み。
・カードの「詳細」＝中身（収録・タグ・メモ）を見る。
・カードの「編集」＝タグやメモの追加・修正・削除。
　メモは「担当者を選ぶ → 書く → 保存」。

【資料の中身や文言を直したいとき】
  そのまま編集しようとせず、私（AI）に「ここをこう直して」と言ってください。
  資料を差し替えたら、一覧のプレビュー画像も作り直してお渡しします。
==========================================================

==========================================================
 メモの同時編集について（安心ポイント）
==========================================================
・6人が同時に書いても、メモ・タグは消えません（1件ずつ別々に保存する設計）。
・約30秒ごとに、画面が静かに最新へ更新されます（入力中・編集中は邪魔しません）。
・保存できなかった時だけ「保存に失敗」と出るので、その時はもう一度押してください。
・GASのコードを新しくしたら、全員ブラウザを「スーパーリロード」して最新に揃える。
   Windows: Ctrl+Shift+R ／ Mac: Cmd+Shift+R
==========================================================
